# 🌸 Flower Knight Girl - 全自动后台周回终端 (v1.0)

本项目是针对网页游戏《Flower Knight Girl》（美少女花骑士）开发的自动化挂机工具。基于 Python + OpenCV + Win32 API 开发，支持真·后台运行，不占用鼠标键盘。

## ✨ 核心特性
- **真·后台挂机**：利用 Win32 后台渲染技术，窗口被完全遮挡（不最小化）仍能正常运行，不影响工作。
- **拟人化操作**：随机点击偏移 + 随机延迟，最大限度降低账号风险。
- **智能避障**：自动检测体力耗尽弹窗，物理隔离“华灵石”按钮，仅消耗免费恢复蜜。
- **现代化界面**：基于 CustomTkinter 的暗色调高级 UI。
- **多屏适配**：完美支持多显示器、不同 DPI 缩放环境。

## 🚀 快速开始（针对普通用户）
1. 在 [Releases] 页面下载最新版本的压缩包并解压。
2. 运行 `main.exe`。
3. 打开游戏并进入关卡，勾选游戏内置的“オート出撃”（自动出击）。
4. 在工具界面下拉选择游戏窗口，设置吃药上限，点击“启动”。
